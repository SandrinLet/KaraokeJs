# 🎤 Karaokê

Este projeto foi desenvolvido durante as aulas de JavaScript que eu estava ministrando. 
Ele simula um karaokê, exibindo a letra da música sincronizada diretamente no terminal mas a ideia é continuar o projeto também para exibição no navegador.


## 🚀 Tecnologias e Conceitos Aplicados
Para construir este projeto, utilizei:

* **Async/Await & Promises:** Para gerenciar o tempo de espera (`timeout`) entre cada verso da música.
* **Manipulação de Arrays de Objetos:** Organização sistemática da letra e dos tempos de exibição.
* **Iteração com `for...of`:** Para percorrer a lista de frases de forma sequencial e organizada.
* **Interface no Terminal:** Uso do `console.clear()` para criar uma experiência dinâmica e limpa para o usuário.
